This guide walks you through the basics of using Filecast. You'll learn how to launch the application, search for apps and files, navigate the four views, and use essential keyboard shortcuts.

<Note>
Make sure you've [installed Filecast](/installation) before starting this guide.
</Note>

## Launch Filecast

<Steps>
  <Step title="Start the application">
    Launch Filecast from your terminal:

    ```bash
    filecast
    ```

    The launcher window appears in the top center of your screen.
  </Step>

  <Step title="Toggle with the global hotkey">
    Press `Super+Space` (Windows/Command key + Space) to hide and show Filecast from anywhere.

    <Note>
    The `Super` key is typically the Windows key on most keyboards, or the Command key on Mac keyboards.
    </Note>

    Try it now:
    1. Press `Super+Space` to hide Filecast
    2. Press `Super+Space` again to bring it back

    This works from any application - you never need to alt-tab to Filecast.
  </Step>
</Steps>

## Search Files and Content

Filecast has powerful search capabilities beyond just launching apps.

### Find Files by Name

Use the `/` prefix to search for files by name:

```
/config      # Find files named "config"
/main.rs     # Find Rust source files
/.bashrc     # Find dotfiles
```

<Note>
If you have `fd` installed, file searches are much faster and automatically respect `.gitignore` files.
</Note>

### Search Inside Files (Grep)

Use the `@` prefix to search file contents:

```
@TODO           # Find all TODO comments
@import React   # Find React imports
@function       # Find function definitions
```

Filecast searches your home directory by default. Configure exclusions in `~/.config/filecast/search.yaml` to skip directories like `node_modules` or `.git`.

### Run Shell Commands

Use the `:` prefix to execute shell commands:

```
:ls -la         # List files
:git status     # Check git status
:pwd            # Print working directory
```

Commands run in your home directory. Command history is saved - use `в†‘`/`в†“` to browse previous commands.

## Explore the Four Views

Filecast has four main views accessible via `Ctrl+1/2/3/4`:

<Tabs>
  <Tab title="Search (Ctrl+1)">
    The default view for searching apps, files, and running commands.

    **Key shortcuts**:
    - `в†‘`/`в†“` - Navigate results
    - `Enter` - Execute/open selected item
    - `Escape` - Clear search, then hide window

    **Search syntax**:
    - `text` - Fuzzy search apps and files
    - `@pattern` - Grep file contents
    - `/pattern` - Find files by name
    - `:command` - Run shell command
  </Tab>

  <Tab title="Files (Ctrl+2)">
    A full file browser with vim-style navigation.

    **Key shortcuts**:
    - `j`/`k` or `в†‘`/`в†“` - Navigate files
    - `l`/`Enter` or `в†’` - Open file / Enter directory
    - `h`/`Backspace` or `в†ђ` - Go to parent directory
    - `r` - Refresh current directory
    - `c` - Enter command mode (run shell commands in current directory)
  </Tab>

  <Tab title="Clipboard (Ctrl+3)">
    Access your clipboard history.

    Filecast automatically tracks everything you copy. Clipboard entries expire after 24 hours unless pinned.

    **Key shortcuts**:
    - `j`/`k` or `в†‘`/`в†“` - Navigate entries
    - `Enter` - Copy selected entry to clipboard
    - `p` - Pin/unpin entry (prevents expiration)
    - `d` - Delete entry

    **Features**:
    - Stores last 50 clipboard entries
    - Automatic expiration after 24 hours
    - Pin important items to keep them permanently
  </Tab>

  <Tab title="Settings (Ctrl+4)">
    Configure Filecast's window position.

    **Available positions**:
    - Top Center (default)
    - Center
    - Top Left / Top Right
    - Bottom Center / Bottom Left / Bottom Right

    Changes take effect immediately.
  </Tab>
</Tabs>

## Common Workflows

### Launch an Application

1. Press `Super+Space`
2. Type the app name (fuzzy matching works: `firef` в†’ Firefox)
3. Press `Enter`

### Find and Open a File

1. Press `Super+Space`
2. Type `/filename` to search by name
3. Press `в†‘`/`в†“` to select, `Enter` to open

Or use the Files view (`Ctrl+2`) for browsing.

### Search Code for a Pattern

1. Press `Super+Space`
2. Type `@pattern` (e.g., `@TODO` or `@import React`)
3. Browse results and press `Enter` to open the file

### Run a Quick Command

1. Press `Super+Space`
2. Type `:command` (e.g., `:git status`)
3. Press `Enter` to execute

### Restore a Clipboard Entry

1. Press `Super+Space`
2. Press `Ctrl+3` to open Clipboard view
3. Use `в†‘`/`в†“` to find the entry
4. Press `Enter` to copy it
